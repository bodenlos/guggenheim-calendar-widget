== Description ==

Guggenheim Event Calendar Demonstration Widget is a WordPress widget/plugin demonstration that displays upcoming events from the calendar at guggenheim.org.


== Installation ==

1. Uncompress and upload the `/guggenheim-calendar-widget/` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in your WordPress admin. panel
